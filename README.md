## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd charishma_portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Sections

- **About**: Introduction, contact information, and education
- **Experience**: Professional work experience with detailed achievements
- **Projects**: Showcase of notable projects with technologies used
- **Skills**: Technical skills organized by category and certifications
- **Contact**: Contact information and social links

## Customization

To customize the portfolio:

1. Update personal information in `src/components/About.jsx`
2. Modify experience details in `src/components/Experience.jsx`
3. Add or update projects in `src/components/Projects.jsx`
4. Adjust skills in `src/components/Skills.jsx`
5. Update colors and styling in CSS files (using CSS custom properties)
